# 《定积分的换元积分法和分部积分法》完整教学方案

## 一、整体教学安排

**课时分配：** 3课时（每课时45分钟）
**教学对象：** 高等数学（微积分）学习者
**教学形式：** 多媒体+板书结合，讲练结合

---

## 二、第一课时：定积分的换元积分法（45分钟）

### （一）课堂引入（5分钟）

**教师语言：**
"同学们好！上节课我们回顾了不定积分的换元法，大家还记得那个经典例子 ∫√(1-x²)dx 是怎么做的吗？"

（等待学生回答）

"对，用三角代换 x = sin t。那么如果我给这个积分加上上下限，比如计算 ∫₀¹ √(1-x²)dx，也就是单位圆在第一象限的面积，该怎么处理呢？还能像不定积分那样先求原函数，再把 t 回代成 x，最后代入上下限吗？"

（稍作停顿，观察学生反应）

"今天我们要学习的定积分换元法，会给出一个更直接、更高效的方法。先给大家看一个关键区别："

（板书）
**不定积分换元：** 换元 → 积分 → 回代
**定积分换元：** 换元+换限 → 积分 → 直接得结果

"注意到区别了吗？定积分换元时，**积分限要跟着变量一起换**，而且最后不需要回代！这就是我们今天要掌握的核心思想。"

### （二）新知讲解：定理1（15分钟）

**教师语言：**
"现在请大家看定理1的完整表述（展示课件）。我来解读一下条件：

第一，f(x)在[a,b]上连续——这是定积分存在的前提，我们之前学过。

第二，我们引入新变量t，设x = φ(t)，它需要满足：
1. **端点对应：** φ(α) = a, φ(β) = b
   （这意味着当t从α变到β时，x正好从a变到b）
2. **良好性质：** φ(t)在[α, β]上有连续导数，且值域包含[a,b]
   （保证换元后函数仍然良好，可以积分）

满足这些条件时，我们有重要的**换元公式**："

（板书，一字一句地）
\[
\int_a^b f(x) dx = \int_{\alpha}^{\beta} f[\varphi(t)] \varphi'(t) dt
\]

**教师语言：**
"请大家仔细观察这个公式的左右两边：
- 左边是对x积分，积分限是a到b
- 右边是对t积分，积分限是α到β
- f(x)变成了f[φ(t)]
- dx变成了φ'(t)dt

**这就是定积分换元法的完整操作：变量、微分、积分限同时更换！**

特别提醒两点（展示课件注记）：
1. 即使φ(t)的值域略微超出[a,b]，只要f(x)在超出部分也连续，结论依然成立
2. 即使α > β，公式也成立，因为定积分上下限交换会变号

现在我们通过例题来具体感受。"

### （三）例题讲解（20分钟）

#### 例1：∫₀ᵃ √(a²-x²) dx (a>0)

**教师语言：**
"这是一个经典问题，表示半径为a的圆在第一象限的面积。看到a²-x²，我们自然想到——"

**学生：** "三角代换！"

**教师：** "很好！设x = a sin t。接下来关键一步：确定新的积分限。
当x=0时，a sin t = 0 ⇒ t = kπ，我们取最简值t=0。
当x=a时，a sin t = a ⇒ sin t = 1 ⇒ t = π/2 + 2kπ，取t=π/2。
所以新的积分限是：t从0到π/2。"

（边板书边讲解）
"接下来：
dx = a cos t dt
√(a²-x²) = √(a²-a²sin²t) = a√(cos²t) = a|cos t|

因为在[0, π/2]上cos t ≥ 0，所以绝对值直接去掉，就是a cos t。

于是原积分 = ∫₀^{π/2} (a cos t)·(a cos t) dt = a² ∫₀^{π/2} cos²t dt"

**教师：** "cos²t怎么积分？用降幂公式：cos²t = (1+cos2t)/2
所以 = a² ∫₀^{π/2} (1+cos2t)/2 dt = (a²/2)[t + (1/2)sin2t]|₀^{π/2}
= (a²/2)·(π/2) = πa²/4

这就是半径为a的圆面积的四分之一。几何意义和计算结果完美吻合！"

---

#### 例3：∫₀^{π/2} cos⁵t sin t dt

**教师语言：**
"这个积分有两种做法，对比一下：

**方法一：显式换元**
令x = cos t，则dx = -sin t dt
当t=0时，x=1；t=π/2时，x=0
原式 = ∫₁⁰ x⁵·(-dx) = ∫₀¹ x⁵ dx = 1/6

**方法二：凑微分（更简洁）**
原式 = -∫₀^{π/2} cos⁵t d(cos t) = -[cos⁶t/6]|₀^{π/2} = 1/6

**重要发现：** 凑微分法本质也是换元，但我们不写出新变量，所以**积分限不变**！这是定积分换元的一个巧妙技巧。"

### （四）要点总结（5分钟）

**教师语言：**
"第一课时的核心要点，请大家牢记：

1. **换元必换限**——新旧积分限通过x=φ(t)对应
2. **微分要换对**——dx = φ'(t)dt，不能漏掉导数
3. **无需回代**——直接计算新积分得出数值结果
4. **凑微分更简**——不显式换元时，积分限不变

记住这四点，定积分换元法就掌握了基础。下节课我们将学习换元法的更多应用技巧。"

## 三、第二课时：换元法的应用与分部积分法（45分钟）

### （一）复习引入（5分钟）

**教师语言：**
"上节课我们学会了换元法的基本操作。现在思考一个问题：计算∫_{-1}¹ x³ dx，你会怎么做？"

（学生可能回答直接积分或利用奇偶性）

"如果直接积分：原函数是x⁴/4，代入得(1/4)-(-1/4) = 0。但有没有更快的方法？观察被积函数x³和区间[-1,1]——x³是奇函数，对称区间上奇函数的积分一定是0！这就是我们今天要学的第一个重要性质。"

### （二）奇偶函数的积分性质（10分钟）

#### 例5：证明奇偶函数在对称区间上的积分性质

**教师语言：**
"设f(x)在[-a,a]上连续。我们把积分拆开：
∫_{-a}ᵃ f(x)dx = ∫_{-a}⁰ f(x)dx + ∫₀ᵃ f(x)dx

对第一个积分作换元：令x = -t，则dx = -dt
当x=-a时，t=a；x=0时，t=0
所以∫_{-a}⁰ f(x)dx = ∫_{a}⁰ f(-t)(-dt) = ∫₀ᵃ f(-t)dt

于是原式 = ∫₀ᵃ f(-t)dt + ∫₀ᵃ f(x)dx = ∫₀ᵃ [f(-x) + f(x)] dx"

**教师：** "分析这个结果：
- 若f是**偶函数**：f(-x) = f(x)，被积函数 = 2f(x) ⇒ 积分 = 2∫₀ᵃ f(x)dx
- 若f是**奇函数**：f(-x) = -f(x)，被积函数 = 0 ⇒ 积分 = 0

这就是著名的**'偶倍奇零'**法则。让我们快速做几个练习："

（快速提问）
1. ∫_{-π}^{π} sin x dx = ? （奇函数，答案：0）
2. ∫_{-2}² (x²+1) dx = ? （偶函数，答案：2∫₀² (x²+1)dx = 28/3）

### （三）区间对称性质（8分钟）

#### 例6：证明∫₀ᵃ f(x)dx = ∫₀ᵃ f(a-x)dx

**教师语言：**
"这个结论很有趣：在[0,a]上，f(x)和它的'镜像'f(a-x)积分相等。
证明很简单：令x = a - t，则dx = -dt
当x=0时，t=a；x=a时，t=0
所以左边 = ∫_{a}⁰ f(a-t)(-dt) = ∫₀ᵃ f(a-t)dt = 右边

**重要推论：** 当a=π/2时，有
∫₀^{π/2} f(sin x)dx = ∫₀^{π/2} f(cos x)dx

比如∫₀^{π/2} sinⁿx dx = ∫₀^{π/2} cosⁿx dx，这个结论在三角函数积分中非常有用。"

### （四）分部积分法引入（7分钟）

**教师语言：**
"现在我们来学习定积分的第二种重要方法——分部积分法。回忆不定积分的分部积分公式："

（板书）∫ u dv = uv - ∫ v du

"定积分的版本非常相似，但多了一个关键步骤："

（板书，加重语气）
**定理2：** ∫ₐᵇ u(x)v'(x)dx = u(x)v(x)|ₐᵇ - ∫ₐᵇ v(x)u'(x)dx

"看，就是在不定积分公式的基础上，把uv部分带上了**上下限**。
u和v的选择原则和不定积分完全一样：**'反对幂三指'**（反三角函数、对数函数、幂函数、三角函数、指数函数），排在前面的优先选为u。"

### （五）分部积分法例题（15分钟）

#### 例13：∫₀^{1/2} arcsin x dx

**教师语言：**
"这个积分含有反三角函数，我们设：
u = arcsin x, dv = dx
则 du = dx/√(1-x²), v = x

代入公式：
原式 = x·arcsin x|₀^{1/2} - ∫₀^{1/2} x/√(1-x²) dx
= (1/2)·(π/6) - 0 - ∫₀^{1/2} x/√(1-x²) dx"

**教师：** "剩下的积分用凑微分：
-∫₀^{1/2} x/√(1-x²) dx = (1/2)∫₀^{1/2} d[√(1-x²)]'（注意符号）
= √(1-x²)|₀^{1/2} = √(1-1/4) - 1 = √3/2 - 1

所以最终结果 = π/12 + √3/2 - 1"

---

#### 例14：∫₀¹ e^{√x} dx

**教师语言：**
"这个积分含有根号，我们先换元去根号：
令 √x = t，则 x = t², dx = 2t dt
当x:0→1时，t:0→1

原式 = ∫₀¹ eᵗ·2t dt = 2∫₀¹ t eᵗ dt

现在用分部积分：设u = t, dv = eᵗ dt
则du = dt, v = eᵗ

原式 = 2[ t eᵗ|₀¹ - ∫₀¹ eᵗ dt ]
= 2[ (1·e - 0) - (e - 1) ]
= 2[ e - e + 1 ] = 2"

**教师：** "这个例题展示了**先换元、再分部**的综合运用思路，很有代表性。"

---

## 四、第三课时：综合应用与瓦里斯公式（45分钟）

### （一）复习与引入（5分钟）

**教师语言：**
"前两节课我们分别学习了换元法和分部积分法。实际解题中，往往需要多种方法组合使用。今天我们就来挑战一些综合性问题，并推导一个重要的积分公式——瓦里斯公式。"

### （二）综合例题解析（20分钟）

#### 例15：已知 f(x) = ∫₁^{x²} (sin t)/t dt，求 ∫₀¹ x f(x) dx

**教师语言：**
"这道题有几个难点：f(x)本身是变限积分，而且要求的是x f(x)的积分。我们一步步分析：

第一步：了解f(x)的性质
由定义，f(1) = ∫₁¹ (sin t)/t dt = 0
根据微积分基本公式，f'(x) = (sin x²)/(x²)·2x = (2 sin x²)/x

第二步：处理目标积分 ∫₀¹ x f(x) dx
看到x f(x)，想到分部积分。设：
u = f(x), dv = x dx
则 du = f'(x)dx, v = x²/2

代入公式：
原式 = (x²/2)f(x)|₀¹ - ∫₀¹ (x²/2)f'(x)dx
= 0 - (1/2)∫₀¹ x²·(2 sin x²)/x dx
= -∫₀¹ x sin x² dx"

**教师：** "第三步：计算剩余积分
-∫₀¹ x sin x² dx = -(1/2)∫₀¹ sin x² d(x²) 
= (1/2)cos x²|₀¹ （注意负号和微分的关系）
= (1/2)(cos1 - cos0) = (1/2)(cos1 - 1)

这道题综合运用了：变限积分求导、分部积分、凑微分，体现了知识的融会贯通。"

### （三）瓦里斯公式推导（15分钟）

#### 例16：证明 Iₙ = ∫₀^{π/2} sinⁿx dx 的递推公式

**教师语言：**
"瓦里斯公式是数学中一个优美而重要的公式。我们来推导它的递推关系。

设 Iₙ = ∫₀^{π/2} sinⁿx dx
用分部积分：设 u = sinⁿ⁻¹x, dv = sin x dx
则 du = (n-1)sinⁿ⁻²x cos x dx, v = -cos x

代入：
Iₙ = -sinⁿ⁻¹x cos x|₀^{π/2} + (n-1)∫₀^{π/2} sinⁿ⁻²x cos²x dx

第一项为0（因为sin0=0, cos(π/2)=0）
第二项中，cos²x = 1 - sin²x，所以：
Iₙ = (n-1)∫₀^{π/2} sinⁿ⁻²x dx - (n-1)∫₀^{π/2} sinⁿx dx
= (n-1)Iₙ₋₂ - (n-1)Iₙ"

**教师：** "整理这个等式：
Iₙ + (n-1)Iₙ = (n-1)Iₙ₋₂
nIₙ = (n-1)Iₙ₋₂
得到递推公式：**Iₙ = [(n-1)/n] Iₙ₋₂**"

#### 分奇偶讨论（边推导边板书）

**教师：**
"**情况1：n为偶数，设n=2m**
I₂ₘ = [(2m-1)/(2m)] I₂ₘ₋₂
继续递推：
I₂ₘ₋₂ = [(2m-3)/(2m-2)] I₂ₘ₋₄
...
I₂ = (1/2) I₀
而 I₀ = ∫₀^{π/2} dx = π/2

所以 I₂ₘ = [(2m-1)/(2m)]·[(2m-3)/(2m-2)]·...·(3/4)·(1/2)·(π/2)"

**教师：**
"**情况2：n为奇数，设n=2m+1**
I₂ₘ₊₁ = [(2m)/(2m+1)] I₂ₘ₋₁
递推：
I₂ₘ₋₁ = [(2m-2)/(2m-1)] I₂ₘ₋₃
...
I₃ = (2/3) I₁
而 I₁ = ∫₀^{π/2} sin x dx = 1

所以 I₂ₘ₊₁ = [(2m)/(2m+1)]·[(2m-2)/(2m-1)]·...·(4/5)·(2/3)·1"

**教师：** "这就是完整的瓦里斯公式！它在概率论、物理、工程中有广泛应用，比如计算正态分布的积分。"

### （四）课堂练习与指导（10分钟）

**教师语言：**
"现在大家尝试几道练习题，我将巡视指导：

1. 计算 ∫₀¹ x²√(1-x²) dx （提示：三角代换）
2. 计算 ∫₀^{π} x sin³x dx （提示：利用对称性）
3. 计算 ∫₀¹ x² ln(1+x) dx （提示：分部积分）

（巡视过程中个别指导）

'第一题，看到1-x²，应该想到什么代换？'
'第二题，注意sin³x在[0,π]上的对称性'
'第三题，ln函数应该选为u还是dv？'"

### （五）课程总结（5分钟）

**教师语言：**
"三节课的学习，我们掌握了定积分计算的两种核心方法：

**一、换元积分法**
- 核心：换元必换限
- 技巧：三角代换、根式代换、凑微分
- 性质：偶倍奇零、区间对称性

**二、分部积分法**
- 公式：∫ₐᵇ u v' dx = uv|ₐᵇ - ∫ₐᵇ v u' dx
- 选u原则：反对幂三指
- 应用：处理乘积形式的被积函数

**三、综合运用**
- 复杂问题往往需要先换元、再分部
- 充分利用函数性质简化计算
- 掌握重要公式如瓦里斯公式

数学学习就像搭积木，每个公式、每种方法都是一块积木。只有熟练掌握每一块，才能搭建起解决复杂问题的能力。希望大家通过练习，真正掌握这些方法！"

2. 综合题：教材P.XX 习题11-15题
3. 思考题：用瓦里斯公式估计π的值

## 五、教学特色与建议

### 1. 语言设计特色
- **启发性提问：** 每个知识点前先提出问题，引发思考
- **对比强调：** 突出定积分与不定积分的区别
- **步骤化讲解：** 复杂问题分解为清晰步骤
- **鼓励性语言：** 对学生的正确思路及时肯定

### 2. 板书设计建议
```
左边1/3：核心公式区
  定理1：∫ₐᵇ f(x)dx = ∫_α^β f[φ(t)]φ'(t)dt
  定理2：∫ₐᵇ u v' dx = uv|ₐᵇ - ∫ₐᵇ v u' dx
  重要性质：偶倍奇零、区间对称

中间1/2：例题演算区
  步骤清晰，重点步骤用彩色粉笔标注

右边1/6：要点总结区
  关键词：换元必换限、反对幂三指、偶倍奇零
```

### 3. 多媒体配合
- 定理部分：用PPT展示完整表述
- 几何意义：用动画展示换元过程中的变量对应关系
- 复杂计算：分步展示，避免一页显示过多内容

### 4. 差异化教学建议
- **基础薄弱学生：** 重点掌握换元法的基本步骤
- **中等水平学生：** 熟练运用奇偶性等简化技巧
- **学有余力学生：** 推导瓦里斯公式，探究其应用

---

这份完整教学方案将课件内容、教案设计和教学语言有机融合，既保证了知识的系统性，又注重教学的可操作性。教师可以根据实际课堂情况灵活调整节奏和深度。
